# OpenRPGMaps - Ultimate RPG Map Editor

## Project Overview
OpenRPGMaps is a high-end, web-based 2D top-down RPG map editor designed for both digital use (VTT) and high-resolution physical printing (300 DPI+). It aims to provide a professional-grade toolset comparable to or exceeding established tools like Inkarnate, Dungeon Painter Studio, and Dungeon Alchemist.

## Vision
To create the most powerful and intuitive RPG map editor on the web, featuring professional auto-tiling, advanced lighting, and seamless asset management, all while maintaining a high standard of visual quality and performance.

## Core Technologies
- **Framework:** React 19 (Vite) + TypeScript
- **Canvas Rendering:** react-konva
- **State Management:** Zustand with Command Pattern (Undo/Redo)
- **Styling:** Tailwind CSS + shadcn/ui
- **Database:** Dexie (IndexedDB) for local storage
- **Export/Import:** JSON, ZIP bundles, High-res Image (PNG/JPG/WebP), VTT (Universal VTT)
